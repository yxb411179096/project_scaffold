# 01 项目总流程：高中英语 AI PPT 生成系统

## 一、项目目标

建设一个本地可部署的高中英语教学 PPT 智能生成系统。教师输入课程信息后，系统自动生成：

1. 教学设计；
2. PPT 页面大纲；
3. 逐页课件内容；
4. 每页授课备注；
5. 教师授课稿；
6. 可编辑 PPTX 文件；
7. 后续可接入教材知识库与真实 AI 模型。

本项目不是普通 PPT 美化工具，而是“高中英语教学课件生产流水线”。

---

## 二、推荐总体流程

```text
教师输入课程需求
        ↓
需求解析
        ↓
生成教学设计
        ↓
生成 PPT 大纲
        ↓
生成逐页内容
        ↓
教学逻辑审查
        ↓
语言质量审查
        ↓
生成结构化 JSON
        ↓
编辑页面人工微调
        ↓
导出 PPTX / DOCX
```

---

## 三、为什么必须先生成 JSON

不要让 AI 直接生成 PPT 文件。原因：

1. AI 直接生成 PPT 不稳定；
2. 后续无法精确编辑某一页；
3. 无法统一模板；
4. 无法进行质量审查；
5. 无法复用内容生成授课稿。

正确做法：

```text
AI 输出结构化 JSON
        ↓
程序读取 JSON
        ↓
套用 PPT 模板
        ↓
生成 PPTX
```

## 四、核心功能模块

### 1. 课件任务管理

功能：
- 新建课件任务；
- 查看历史任务；
- 编辑任务信息；
- 删除任务；
- 复制任务重新生成。

字段：
- 年级；
- 教材版本；
- 单元；
- 课题；
- 课型；
- 课时；
- 学生水平；
- 课件风格；
- 补充要求。

---

### 2. 教学设计生成

生成内容：
- Teaching Objectives；
- Key Points；
- Difficult Points；
- Teaching Methods；
- Teaching Procedures；
- Core Competence；
- Assessment。

---

### 3. PPT 大纲生成

按课型生成固定结构。

阅读课推荐结构：
1. Cover
2. Learning Objectives
3. Lead-in
4. Prediction
5. Fast Reading
6. Careful Reading
7. Vocabulary Focus
8. Language Points
9. Group Discussion
10. Summary
11. Homework
12. Blackboard Design

语法课推荐结构：
1. Cover
2. Learning Objectives
3. Observe and Discover
4. Grammar Rule
5. Guided Practice
6. Controlled Practice
7. Communicative Practice
8. Error Correction
9. Summary
10. Homework

写作课推荐结构：
1. Cover
2. Learning Objectives
3. Topic Lead-in
4. Writing Task Analysis
5. Useful Expressions
6. Structure Building
7. Sample Appreciation
8. Guided Writing
9. Peer Review
10. Homework

---

### 4. 逐页内容生成

每页必须包含：

```json
{
  "slide_index": 1,
  "slide_type": "cover",
  "title": "Unit 3 Sports and Fitness",
  "visible_content": ["Reading and Thinking", "Living Legends"],
  "teacher_notes": "Greet students and introduce the topic.",
  "teaching_purpose": "Activate students' background knowledge.",
  "estimated_time": "2 minutes",
  "interaction_type": "Teacher-student interaction"
}
```

---

### 5. 编辑与再生成

必须支持：
- 编辑单页标题；
- 编辑页面内容；
- 编辑授课备注；
- 重新生成某一页；
- 降低难度；
- 提高难度；
- 改成公开课风格；
- 增加互动活动。

---

### 6. 文件导出

第一阶段：
- PPTX 导出；
- DOCX 授课稿导出。

第二阶段：
- PDF 导出；
- Markdown 导出；
- Marp 幻灯片导出。

---

## 五、开发优先级

### 阶段 1：MVP 闭环

目标：先跑通完整流程。

功能：
1. 新建课件；
2. mock 生成课件 JSON；
3. 编辑页面；
4. 导出 PPTX；
5. 导出 DOCX。

### 阶段 2：接入 AI

功能：
1. 接入 Ollama；
2. 接入 DeepSeek / OpenAI；
3. 支持重新生成单页；
4. 支持 AI 修改建议。

### 阶段 3：知识库

功能：
1. 上传教材 PDF / Word；
2. 自动解析文本；
3. 建立向量库；
4. 生成课件时检索教材内容。

### 阶段 4：模板系统

功能：
1. 上传 PPT 模板；
2. 选择课型模板；
3. 支持封面模板；
4. 支持统一字体、颜色、布局。

### 阶段 5：产品化

功能：
1. 用户登录；
2. 教师个人课件库；
3. 历史版本；
4. 公开课精品模板；
5. 分享与下载。
