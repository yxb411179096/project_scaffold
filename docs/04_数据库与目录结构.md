---

## 二、核心数据表

### lesson_tasks

保存课件任务。

字段建议：

```text
id
course_title
grade
textbook
unit
lesson_type
duration
student_level
style
extra_requirements
status
created_at
updated_at
```

---

### ppt_slides

保存每一页课件。

```text
id
task_id
slide_index
slide_type
title
visible_content_json
teacher_notes
teaching_purpose
estimated_time
interaction_type
created_at
updated_at
```

---

### teaching_materials

保存教材资料。

```text
id
title
grade
textbook
unit
file_path
parsed_text
created_at
```

---

### ppt_templates

保存 PPT 模板。

```text
id
name
template_type
file_path
description
created_at
```

---

## 三、JSON 标准

每页 slide 使用统一结构：

```json
{
  "slide_index": 1,
  "slide_type": "lead_in",
  "title": "Lead-in",
  "visible_content": [
    "Look at the pictures.",
    "What sport do you like best? Why?"
  ],
  "teacher_notes": "Ask students to discuss in pairs for one minute.",
  "teaching_purpose": "Activate background knowledge and arouse interest.",
  "estimated_time": "3 minutes",
  "interaction_type": "Pair work"
}
```
