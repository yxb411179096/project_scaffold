# 03 本地部署流程

## 一、本地开发环境

推荐使用 Python 3.10 或 3.11。

```bash
cd english_ppt_ai
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

浏览器访问：

```text
http://127.0.0.1:5000
```

---

## 二、Ollama 本地模型

安装 Ollama 后，默认本地 API 地址为：

```text
http://localhost:11434/api
```

常用命令：

```bash
ollama pull qwen3:30b
ollama run qwen3:30b
```

测试接口：

```bash
curl http://localhost:11434/api/chat -d '{
  "model": "qwen3:30b",
  "messages": [
    {"role": "user", "content": "Generate a senior high English lesson outline."}
  ],
  "stream": false
}'
```

---

## 三、宝塔部署建议

服务器目录示例：

```text
/www/wwwroot/english_ppt_ai
```

部署步骤：

1. 上传项目；
2. 创建虚拟环境；
3. 安装依赖；
4. 配置 `.env`；
5. 使用宝塔 Python 项目管理器或 uWSGI 启动；
6. Nginx 反向代理；
7. 配置 uploads / exports 目录权限。

---

## 四、生产环境注意事项

1. 不要使用 Flask debug 模式；
2. API key 必须写入环境变量；
3. exports 目录要定期清理；
4. 上传文件需要限制大小；
5. 生成任务后期建议改为异步队列；
6. 如果接入公网，需要登录权限；
7. 学生或教师资料要注意隐私。
