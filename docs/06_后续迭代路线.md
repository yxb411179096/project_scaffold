# 06 后续迭代路线

## 1.0 MVP

- 新建课件任务；
- mock 生成 slides；
- 编辑 slides；
- 导出 PPTX；
- 导出 DOCX。

## 1.1 接入本地 AI

- Ollama；
- JSON 输出；
- 失败回退 mock；
- 单页重新生成。

## 1.2 接入云端 AI

- DeepSeek；
- OpenAI；
- 不同模型切换；
- 精品课件模式。

- 解析 PDF / DOCX；
- 检索相关内容；
- 根据教材生成课件。
