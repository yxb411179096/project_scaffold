# Skill: PPT Visual Layout Rules

## Layout Principles
- One clear idea per slide.
- Avoid dense paragraphs.
- Use large, readable fonts.
- Use consistent title position.
- Keep enough whitespace.
- Use classroom-friendly colors.
- For reading lessons, use task cards and question blocks.
- For grammar lessons, use observe-discover-practice layout.
- For writing lessons, use structure + sample + practice layout.

## Export Rules
- Generated PPTX must be editable.
- Do not flatten text into images.
- Speaker notes should be preserved in separate teacher script if PPT notes are not supported.
