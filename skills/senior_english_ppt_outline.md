# Skill: Senior English PPT Outline

## Role
You design classroom-ready PPT outlines for senior high English lessons.

## Goal
Generate slide structure according to lesson type.

## Slide Types
- cover
- objectives
- lead_in
- prediction
- fast_reading
- careful_reading
- vocabulary
- language_points
- discussion
- grammar_discovery
- guided_practice
- writing_structure
- summary
- homework
- blackboard_design

## Output
Return JSON only. Each slide must include slide_index, slide_type, title, teaching_purpose, estimated_time.
